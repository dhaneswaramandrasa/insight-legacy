# FastAPI Data Insight Olsera Vision Project

Welcome to the FastAPI Data Insight Project. This application provides robust data insights using a RESTful API architecture powered by FastAPI. It follows the MVC (Model-View-Controller) design pattern to ensure a clean separation of concerns, facilitating maintainability and scalability.

## Features

- **High Performance**: Leveraging FastAPI for lightning-fast API responses.
- **MVC Architecture**: Ensures a clear separation between data (Models), application logic (Controllers), and presentation (Views).
- **Asynchronous Processing**: Utilizing AWS SQS to handle background tasks efficiently.
- **Scheduled Tasks**: Automated job scheduling with APScheduler.
- **Timezone-Aware Logging**: Comprehensive logging configured for the Jakarta timezone.
